# lab01b-github-markdown

1) First, create a repo with a README.me on Github
1) Next, copy the URL for the repo and add to your terminal with `$ git clone`
1) `cd` to the repo
1) Use `mkdir` `touch` and `cd` to create this file tree:
```
   src
     |__assets
        |__style.css
        |__app.js
     |__index.html
    README.md
    greeting.md
```
